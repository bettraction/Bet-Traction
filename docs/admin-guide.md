# Admin Guide

## Getting Started

### Accessing Admin Dashboard

1. Navigate to https://admin.bettraction.com
2. Login with your admin credentials
3. Complete 2FA if enabled

## Dashboard Features

### Analytics Overview

- Total users
- Total bets
- Total volume
- Active bets
- Pending deposits
- Pending payouts
- Revenue
- Charts and graphs

### Pending Deposits

- List of pending deposits
- Verify on-chain transactions
- Confirm deposits
- Reject invalid deposits
- View transaction details on BaseScan

### Pending Payouts

- List of pending payouts
- Review payout requests
- Process payouts
- Multiple approval levels
- View transaction history

### Bets

- View all bets
- Filter by status
- Resolve disputes
- Cancel bets
- View bet details
- Audit history

### Users

- View all users
- Ban/unban users
- Reset user passwords
- View user activity
- Edit user profiles
- View user balances

### Reports

- Export data to CSV
- Generate financial reports
- User activity reports
- Bet history reports
- Custom date ranges

### Disputes

- View open disputes
- Review evidence
- Make decisions
- Communicate with users
- Document resolutions

### Notifications

- Send announcements
- Send user-specific notifications
- Notification templates
- Scheduled notifications

### Audit Logs

- View all admin actions
- Filter by admin, action, date
- Export logs
- Monitor suspicious activity

## Security Best Practices

### Account Security

- Use strong passwords
- Enable 2FA
- Don't share credentials
- Log out when not using
- Use dedicated admin devices

### Operational Security

- Review pending items daily
- Document all decisions
- Double-check payout addresses
- Verify transactions on BaseScan
- Keep software updated

## Workflows

### Resolving a Bet

1. Navigate to Bets page
2. Find the disputed bet
3. Review all evidence
4. Click "Resolve Bet"
5. Select winner
6. Add resolution notes
7. Confirm

### Processing a Payout

1. Navigate to Pending Payouts
2. Review payout request
3. Verify bet was resolved correctly
4. Click "Process Payout"
5. Confirm recipient address
6. Sign and send transaction
7. Verify on BaseScan

### Banning a User

1. Navigate to Users page
2. Find the user
3. Click "Ban User"
4. Select reason
5. Add notes
6. Confirm
