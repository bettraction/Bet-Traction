# Roadmap

This document outlines the development roadmap for BetTraction.

## Current Status: MVP

**Completed:**
- ✅ Landing Page
- ✅ Wallet Connection (RainbowKit)
- ✅ Create Bet
- ✅ Accept Bet
- ✅ Deposit Flow
- ✅ Payout Flow
- ✅ Basic Analytics
- ✅ Admin Dashboard
- ✅ Vault Transparency
- ✅ PostgreSQL Database
- ✅ Docker Deployment
