# Wallets

## Supported Wallets

BetTraction uses RainbowKit, which supports:

- MetaMask
- Coinbase Wallet
- WalletConnect
- Rainbow
- Trust Wallet
- And many more!

## Network Configuration

### Base Mainnet

- **Chain ID**: 8453
- **RPC URL**: https://mainnet.base.org
- **Explorer**: https://basescan.org
- **Native Currency**: ETH

### Base Sepolia (Testnet)

- **Chain ID**: 84532
- **RPC URL**: https://sepolia.base.org
- **Explorer**: https://sepolia.basescan.org
- **Native Currency**: ETH

## Adding Base to Your Wallet

### MetaMask

1. Open MetaMask
2. Go to Settings → Networks → Add Network
3. Enter:
   - Network Name: Base Mainnet
   - New RPC URL: https://mainnet.base.org
   - Chain ID: 8453
   - Currency Symbol: ETH
   - Block Explorer URL: https://basescan.org
4. Click "Save"

## Connecting Your Wallet

1. Click "Connect Wallet" on the BetTraction website
2. Select your wallet from the list
3. Approve the connection in your wallet
4. Sign the login message

## Security Best Practices

- Use a hardware wallet (Ledger, Trezor) for large amounts
- Never share your seed phrase
- Enable 2FA where available
- Verify URLs before connecting
- Keep your wallet software updated
