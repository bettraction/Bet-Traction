# Changelog

All notable changes to BetTraction will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-03
### Added
- Monorepo structure with Turbo
- Backend API with NestJS
- Web app with React + Vite + RainbowKit
- Admin dashboard
- Docker and docker-compose config
- CI/CD with GitHub Actions
- Comprehensive documentation in docs/ directory
- UI component library (@bettraction/ui)
- Complete database entities
- Authentication with SIWE (Sign-In with Ethereum)
- Vault system for transparent deposits/payouts
